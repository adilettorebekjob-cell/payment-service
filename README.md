# 💳 Payment Service

Простой платежный сервис на **Spring Boot 4.0.5** с REST API для создания, проверки статуса, подтверждения и отмены платежей.

## 🚀 Запуск проекта

1. Убедитесь, что установлена **Java 21+** и **Maven 3.8+**
2. Склонируйте репозиторий
3. Запустите приложение:
   ```bash
   mvn spring-boot:run
